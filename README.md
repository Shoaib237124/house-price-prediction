# 🏠 Bengaluru House Price Prediction

This project predicts housing prices in Bengaluru based on user inputs like location, square footage, number of bedrooms, and bathrooms. It uses a Linear Regression model trained on cleaned Kaggle data and provides predictions through a simple web interface.

---

## 📊 Features

- Built using **Python**, **Flask**, **HTML/CSS/JavaScript**
- Data collected and cleaned from [Kaggle Dataset](https://www.kaggle.com/datasets/sumaiyaarfin/bangalore-house-price-prediction)
- Outlier removal and feature engineering
- Model selection using `GridSearchCV`
- Web interface for interactive prediction
